Question-1: What are the different ways to select an element in the DOM?

Elements in the DOM can be selected using methods like getElementById, getElementsByClassName, getElementsByTagName, querySelector, and querySelectorAll.

Question-2: What is the difference between innerHTML, innerText, and textContent?

innerHTML returns HTML content, innerText returns visible text (excluding hidden), and textContent returns all text including hidden, without HTML formatting.

Question-3: What is event delegation in the DOM?

Event delegation is a technique where a single event listener is added to a parent element to handle events from its child elements using event bubbling.

Question-4: What is event bubbling in the DOM?

Event bubbling is the process where an event starts from the target element and bubbles up through its ancestors in the DOM hierarchy.

Question-5: How do you create, add, and remove elements using JavaScript?

You can create elements using document.createElement(), add them with appendChild() or append(), and remove them using removeChild() or remove().
